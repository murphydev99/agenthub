How It Works

This page demonstrates the AgentHub chatbot embedded using a secure, domain-restricted token. The token is tied to localhost:5174 and will only work when loaded from this domain.

Security Features

  • Domain Validation: The API validates that requests come from the authorized domain
  • Token Expiration: Tokens can be set to expire after a certain period
  • Origin Checking: Cross-origin requests from unauthorized domains are blocked
  • Subdomain Support: Tokens work for subdomains of the authorized domain
