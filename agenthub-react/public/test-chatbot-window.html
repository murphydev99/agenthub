Features

  • ✅ Always visible window (no toggle button)
  • ✅ Responsive sizing based on viewport
  • ✅ Configurable position (6 options)
  • ✅ Custom dimension support
  • ✅ Automatic token authentication
  • ✅ Mobile responsive
  • ✅ Programmatic API control

Integration Code

Add this script to your website to embed the always-visible chatbot:

<script> // Optional: Configure the widget window.agentHubConfig = { position: 'bottom-right', // or 'bottom-left', 'top-right', 'top-left', 'center', 'fullscreen' maxWidth: 450, maxHeight: 650, minWidth: 320, minHeight: 400, margin: 20 }; </script> <script src="/chatbot-widget-window.js"></script>

Programmatic API

After loading, you can control the widget programmatically:

// Change position window.agentHub.setPosition('center'); // Set custom dimensions window.agentHub.setDimensions(400, 600); // Trigger auto-resize window.agentHub.resize();
